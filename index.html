Web of Data

Universal Open Protocol and Data Representation

Specification Software

The Concept

A universal api and common data representation that can be used across the web and within organisations. Rather than many bespoke APIs and clients libraries, a single common, universal means to expose, consume, update and query data. Layered on top of this protocol is a data representation that uses global identifiers for things and schema. The Web Of Data specification is here. It`s simple, powerful and supported by open source implementations.

The Specification

Pulling together 20 years of standards work in the area of data interchange formats and APIs. The WoD specification brings together all the best things from OData, RDF, Topic Maps, and SDShare; to deliver a easy to use, easy to implement, web scale, universal data representation and protocol.

The specification is developed under an open process on github. The latest version of the specification is 0.8

The protocol part of the specification is also published as swagger interface.

Software

The WoD reference implementation (MIT license) provides a production ready, small footprint, high performance WoD server. This can be downloaded and used for publishing WoD data. In addition, community developed open source components provide different kinds of WoD compliant software.

The software page contains links for downloads and links to open source projects.
